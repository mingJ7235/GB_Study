클래스()

	클래스는 타입이다. 

	공통요소를 매번 새로 선언하지 않고 
	하나의 클래스 안에서 한 번 선언하여
	가져다 사용한다. 

객체 (인스턴스 변수)
	추상화된 개념을 구체화 시키는 것.
	클래스의 필드에 접근할 수 있는 변수

	값이 담기면 변수, 주소값이 담기면 객체!!

객체화 (인스턴스  = instance -> 예시. 즉, 예시를 들어서 구체화를 한다는 느낌으로 알면된다. )
	Car mom = new Car();

	Car() 의역할 : heap 메모리에 mom을 할당하고, 
		다시 그 할당된 필드의 주소값을 가져온다! 
	그러므로 new Car(); 부분은 값이다!!

	new를 만나서 heap 메모리에 할당을 한다. 
	할당된 필드의 주소값을 가져온다. 
	Car() 메소드의 리턴값은 주소값이다. 
	자동으로 리턴이된다. 
	필드를 메모리에 할당을 하고 주소값을 가져온다. 
	그 주소값이 mom으로 대입되는 것이다 !

	그렇기 때문에 .(하위연산자, 멤버변수접근 연산자) 를 사용하여
	mom 객체의 주소로 접근하는것이다!!

※ . : 하위연산자, 멤버변수 접근 연산자 
	=> mom. 으로하면 메모리에 할당된 필드의 주소로 가게된것이다. 
	mom.필드명 으로 접근한다.	

	즉, 객체화란 객체를 만드는 작업이다.
	추상적인 개념을 구체화 시키는 작업을 말한다. 
	Car	mom	=	new 	car();
	클래스	객체		heap	할당된 필드의 주소값


생성자 (Constructor)
	형태 : 클래스명()

	이름 뒤에 소괄호가 있기 때문에 메소드와 동일한 기능을 가지고 있다.
	하지만 생성자는 리턴이라는 기능을 제공하지 않기 때문에 메소드라고 부르면 안된다.
	
	이미 할당된 주소값을 리턴해야하기 때문에 리턴기능을 사용할 수가없다.

	
생성자의 목적
	1. 해당 클래스 필드를 메모리에 할당한 후 할당된 주소값을 리턴한다.
	2. 초기화의 목적이 있다.
