면접 강의 bit와 byte

1byte는 정보의 최소 단위

boolean 논리형 1byte 
	-> 전구 1개만있으면 되는데 왜 8개나 필요하냐? (1byte는 8bit)
	-> 1byte가 정보의 최소단위기 때문에 1byte로 사용되는 것이다.

전구 8개 = 1byte
전구 1개는 bit 즉, 8bit = 1byte다 

그러면, sql에서 NUMBER와 NUMBER(a,b) 의 차이는 무엇인가

NUMBER : 부동 소수점 
	소수점 자리수가 고정되어 있지 않다.
	만약 해당 비트를 넘어가면 반올림된다. 

NUMBER(a,b): 고정 소수점
	소수점 자리 수가 고정되어 있다.
	정수부에 더 큰 정수를 담을 수 있다. 

float 4byte (32bit) 


최상위 비트는 부호비트! 
컴퓨터는 덧샘밖에 없다. 그렇기때문에 음수랑 양수를 비교하기 위해서
최상위 비트가 부호비트다. 0이면 양수, 1이면 음수
12 -> 0 00000000000000000000000000000001100
	=> 맨처음 0이 양수

※실수의 오류.
	실수를 계산할때 비트의 자리수가 부족해서 반올림이 되어 
	실수의 계산이 오류가 나게 된다. 
	부동소수점을 사용했을때 나오는 오류.
