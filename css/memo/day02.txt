캐스캐이딩 (Cascading)
	요소는 하나 이상의 CSS 선언에 영향을 받을 수 있다. 
	이 때 충돌을 피하기 위해서 CSS적용 우선 순위가 필요하다. 

1. 중요도 : CSS가 어디에 선언되었는 지에 따라서 우선순위가 달라진다. 
	(1) 인라인 스타일 (HTML요소 내부에 style 속성으로 사용)	
	ex) <strong sytle = "color:black;"> 
	-> 외부 css파일이나 내부 스타일 시트 sytle 태그에 속하지 않고
		요소 내부에서 속성!으로사용
	-> 가장 우선순위가 높다. 가장 적용할 곳에 가깝기때문에
	(2) 내부 스타일 시트 (HTML문서의 style 태그 안에 위치)
	(3) 외부 스타일 시트 (CSS파일을 따로 만들어서 불러오는 방법)
	(4) 웹 브라우저 기본 스타일
	-> default 즉, 아무 스타일 안줄때

	※적용될 것에서 가까울수록 중요도가 높다!


2. 명시도 : 명확하게 특정할 수록 우선순위가 높아진다. 
	(1) !important (세미콜론찍기전에 선언)
		-> 가장 높음! 
	(2) 인라인 스타일 
	(3) 아이디 선택자 
	(4) 클래스 선택자 
	(5) 태그 선택자 
	(6) 전체 선택자 
	(7) 상속받은 속성 

	!impartante를 이길 수 있는 스타일은 없다. 

3. 선언순서 : 동일한 우선순위의 선택자라면, 나중에 선언된 스타일이 우선 적용된다. 
	여러 개의 클래스 명을 사용하는 경우에도 나중에 선언된 스타일이
	우선 적용된다.

font-size 속성값
	1. 절대적 크기 (px)
		텍스트의 크기를 명시된 크기 그대로 설정하고자 할 때 사용.
		절대적 크기로 설정된 텍스트는 모든 브라우저에서 같은 크기로 표현된다.

	2. 상대적 크기
		텍스트를 둘러싸고 있는 HTML 요소들의 크기에 따라 텍스트의
		크기도 같이 변하는 방식이다. 또한, 사용자가 웹 브라우저를 통해
		텍스트의 크기를 직접 변경할 수도 있다. 


크기 단위 (기본 크기 : 16px)
	- 백분율 단위 (%)
		기본 크기를 100%로 놓고, 그에 대한 상대적인 크기 설정.

	- 배수단위 (em)
		부모 요소에 중첩되어 1.0이 100%, 1.5는 150%
		상대적인 크기 설정	(ex> 1.5em)
		부모사이즈를 기준으로 1.5배다 라는 의미임

	- 배수단위 (rem)
		부모에 상관없이 HTML 기본 크기에 적용되어 상대적인 크기 설정.
		즉 기본 크기 16px를 기준으로 상대적 크기를 설정함. 

	- 픽셀 단위 (px)
		스크린의 픽셀을 기준으로 하는 절대적인 크기를 설정한다.
